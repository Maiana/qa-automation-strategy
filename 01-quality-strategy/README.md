# QA Automation Strategy Portfolio

Author: Maiana Alebrant

## Overview

This repository demonstrates my approach to Quality Engineering for a B2C e-commerce platform operating in the European market.

The goal of this project is to showcase:

- Strategic test planning
- Risk-based prioritization
- Scalable automation architecture
- CI/CD quality integration
- Meaningful quality metrics

This portfolio reflects not only automation skills, but also product-oriented quality thinking.

---

## Simulated Product Context

The system represents a B2C e-commerce platform with:

- Weekly releases
- Revenue-critical checkout flow
- Authenticated user accounts
- Public product catalog
- REST API backend
